{\bfseries{raylib is a simple and easy-\/to-\/use library to enjoy videogames programming.}}

raylib is highly inspired by Borland BGI graphics lib and by XNA framework and it\textquotesingle{}s especially well suited for prototyping, tooling, graphical applications, embedded systems and education.

{\itshape NOTE for ADVENTURERS\+: raylib is a programming library to enjoy videogames programming; no fancy interface, no visual helpers, no debug button... just coding in the most pure spartan-\/programmers way.}

Ready to learn? Jump to \href{https://www.raylib.com/examples.html}{\texttt{ code examples!}}

\begin{DoxyItemize}
\item {\bfseries{NO external dependencies}}, all required libraries are \href{https://github.com/raysan5/raylib/tree/master/src/external}{\texttt{ included into raylib}}
\item Multiple platforms supported\+: {\bfseries{Windows, Linux, Mac\+OS, RPI, Android, HTML5... and more!}}
\item Written in plain C code (C99) using Pascal\+Case/camel\+Case notation
\item Hardware accelerated with Open\+GL\+: {\bfseries{1.\+1, 2.\+1, 3.\+3, 4.\+3, ES 2.\+0, ES 3.\+0}}
\item {\bfseries{Unique Open\+GL abstraction layer}} (usable as standalone module)\+: \href{https://github.com/raysan5/raylib/blob/master/src/rlgl.h}{\texttt{ rlgl}}
\item Multiple {\bfseries{Fonts}} formats supported (TTF, OTF, FNT, BDF, sprite fonts)
\item Multiple texture formats supported, including {\bfseries{compressed formats}} (DXT, ETC, ASTC)
\item {\bfseries{Full 3D support}}, including 3D Shapes, Models, Billboards, Heightmaps and more!
\item Flexible Materials system, supporting classic maps and {\bfseries{PBR maps}}
\item {\bfseries{Animated 3D models}} supported (skeletal bones animation) (IQM, M3D, gl\+TF)
\item Shaders support, including model shaders and {\bfseries{postprocessing}} shaders
\item {\bfseries{Powerful math module}} for Vector, \doxylink{structMatrix}{Matrix} and Quaternion operations\+: \href{https://github.com/raysan5/raylib/blob/master/src/raymath.h}{\texttt{ raymath}}
\item Audio loading and playing with streaming support (WAV, QOA, OGG, MP3, FLAC, XM, MOD)
\item {\bfseries{VR stereo rendering}} support with configurable HMD device parameters
\item Huge examples collection with \href{https://github.com/raysan5/raylib/tree/master/examples}{\texttt{ +140 code examples}}!
\item Bindings to \href{https://github.com/raysan5/raylib/blob/master/BINDINGS.md}{\texttt{ +70 programming languages}}!
\item {\bfseries{Free and open source}}
